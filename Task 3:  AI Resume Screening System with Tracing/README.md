
## 🚀 Overview
This project is an AI-powered Resume Screening System that evaluates candidates based on a job description.

## 🧠 Features
- Extracts skills from resume
- Matches with job requirements
- Calculates score (0–100)
- Provides explanation for the score

## 🔄 Pipeline
Resume → Skill Extraction → Matching → Scoring → Explanation

## ⚙️ Tech Stack
- Python
- LangChain
- HuggingFace
